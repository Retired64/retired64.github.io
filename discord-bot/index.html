Discord Bot en Termux - Resumen

Descripción

Este repositorio proporciona un script que permite la instalación y ejecución de un bot de Discord directamente en Termux, un emulador de terminal para Android. El script está diseñado para facilitar la configuración y ejecución del bot sin necesidad de conocimientos avanzados de programación.

Instalación

Para instalar el bot de Discord en Termux, sigue estos pasos:

  1. actualiza termux:
    2. yes|pkg update && yes|pkg upgrade
  2. instala los requisitos:
    3. pkg install nodejs
  3. Módulos Necesarios
    4. npm install discord.js
    npm install googleapis
  4. Instalar el servidor:
    5. npm install -g pm2

Ejecución

Una vez instalado, puedes iniciar el bot con el siguiente comando Suponiendo que ti archivo del bot está en el directorio actual:

pm2 start tu-bot.js --name "tu_bot"

Donde tu-bot.js es el archivo de tus comandos de bot y el que está entre comillas es un nombre para identificar el proceso del bot pueda ser el que sea. El bot se conectará a tu servidor de Discord automáticamente y estará listo para recibir comandos.

Detener el bot

Comandos después de iniciar tu bot:

Lista de tu proceso de bot

pm2 list

Detener el proceso de tu bot discord que guardaste en el comando --name "tu_bot"

pm2 stop tu_bot

Donde tu_bot es el nombre del servidor que creaste

Datos Importantes

El bot Podría detenerse y dejar de funcionar en discord para ello requiere que no desconectes tu wifi actual ya que si es cambiado el bot tendrá que ser detenido e iniciado de vuelta, también podría detenerse si el proceso de termux es interrumpido por ejemplo la optimización de batería deberías hacer que no se optimice para ello pon la aplicación sin restricciones de uso de batería por ejemplo y activar el comando:

  • termux-wake-lock - para mantener a termux activo deanera forzada
  • termux-wake-unlock - para poder quitar termux y cerrarlo después que quieras apagar tu bot... ojo que esto es una ayuda extra más lo importante es lo de arriba de esto del wifi

Conclusión

Este bot de Discord es una excelente forma de automatizar tareas y añadir funcionalidades a tu servidor de Discord desde un dispositivo Android utilizando Termux. La instalación y configuración son sencillas, y el código es fácilmente personalizable para adaptarse a tus necesidades específicas.
